星甘StarGantt是一款专业的项目进度管理工具 ，致力于打造简洁易用的在线甘特图制作平台。无论你是项目经理、产品经理，还是需要管理个人项目的自由职业者，都值得试试这款工具。

**在线使用**：[http://g.openauth.net.cn](http://g.openauth.net.cn)

### 💡 为什么选择星甘？

#### 🎯 专业而直观的甘特图体验

星甘提供了媲美Microsoft Project的甘特图功能：

- **多视图模式**：默认视图、月视图、季度视图，满足不同项目周期需求
- **任务依赖管理**：支持前置任务设置，自动生成依赖关系线
- **进度可视化**：实时显示任务完成进度，项目状态一目了然
- **拖拽排序**：支持任务拖拽重新排序，操作简单直观

#### 🏢 类Outlook的现代化界面

界面设计借鉴了Microsoft Outlook的设计语言，看起来既专业又熟悉：

- **品牌区域**：左侧显示项目信息，支持在线编辑项目名称、编码、描述
- **工具栏**：中间是各种操作按钮，功能丰富但布局清晰
- **用户中心**：右侧用户信息区，支持登录和个人设置

#### 📊 强大的任务管理功能

每个任务都可以设置丰富的属性：

- **基础信息**：任务名称、开始时间、工期、完成进度
- **状态管理**：未开始、进行中、已完成、已暂停、已取消
- **人员管理**：负责人、相关方字段
- **依赖关系**：前置任务设置，避免任务冲突
- **任务描述**：详细的任务说明

#### 🔧 实用的辅助功能

- **字段自定义**：可以选择显示/隐藏不同的字段列
- **数据导出**：支持导出Excel格式，方便汇报和备份
- **项目切换**：支持多项目管理，快速切换不同项目
- **收藏功能**：可以收藏重要项目，方便快速访问

#### 📝 使用场景举例

星甘特别适合以下场景：

1. **软件开发项目**：需求分析→设计→开发→测试→上线的完整流程管理
2. **活动策划**：从前期准备到活动执行的全流程规划
3. **工程项目**：建筑、装修等需要严格时间控制的项目
4. **旅行规划**：快速规划旅行路线
5. **学习计划**：考研、考证等长期学习目标的时间规划
